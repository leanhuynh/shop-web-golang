**Chủ đề**

- Ứng dụng mua sắm

**Các tính năng nổi bật**

- Forgot-password

- Authentication

- Tìm kiểm sản phẩm (theo keyword, tag, category, giá, ngày)

- Xác nhận mua hàng (gửi email xác nhận)

**Công nghệ sử dụng**

- Golang

- HTML, CSS

- JWT

- MariaDB
